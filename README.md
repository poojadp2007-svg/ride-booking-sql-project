# 🚗 Ride Booking SQL Project

## 📌 Description
This project is a Ride Booking System Database built using SQL.  
It manages data for users, drivers, rides, and payments.

It simulates a real-world cab booking system similar to Uber.

---

## 🎯 Features
- User management
- Driver details with ratings
- Ride booking and tracking
- Payment system
- Data analysis using SQL queries

---

## 🧱 Tables Used
- Users
- Drivers
- Rides
- Payments

---

## 💻 SQL Concepts
- CREATE TABLE
- INSERT INTO
- SELECT queries
- JOIN
- GROUP BY
- Aggregate functions

---

## 📊 Sample Queries
- Total rides
- Driver earnings
- Completed rides
- Most active driver
- Revenue analysis

---

## 🚀 Project Structure
- schema.sql
- insert_data.sql
- queries.sql
